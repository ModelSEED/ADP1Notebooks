# ADP1 Notebooks

Jupyter notebooks for ADP1 (Acinetobacter baylyi) genome analysis and modeling.

## Notebooks

The `notebooks/` directory contains the following analysis notebooks:

- **ADP1AIGeneAssociation.ipynb** - AI-based gene association analysis
- **ADP1AIModelCompare.ipynb** - Model comparison using AI approaches  
- **ADP1AnnotationAnalysis.ipynb** - Genome annotation analysis
- **ADP1DGOAAnalysis.ipynb** - DGOA (Directed Gene Ontology Analysis) 
- **ADP1ModelReview.ipynb** - Metabolic model review and validation

## Data

- `notebooks/data/` - Input data files including proteomics and published models
- `notebooks/datacache/` - Cached processed data
- `notebooks/nboutput/` - Notebook output files

## Usage

```bash
jupyter notebook notebooks/
```
